## What I'm Building

### JustSearch

`JustSearch` is a Linux-first search and retrieval platform designed to help teams modernize legacy search stacks without throwing away the systems they already depend on.

The direction:

- hybrid search over existing enterprise data
- vector and semantic workflows layered onto legacy search infrastructure
- `JustSearch Signal`: a late-interaction retrieval and reranking layer for stronger second-stage relevance
- secure local AI options for high-trust environments
- operator-friendly tooling for migration, tuning, testing, and runtime packaging

In plain English: I like taking messy, powerful, under-loved systems and turning them into clean, practical tools that people can actually operate.
